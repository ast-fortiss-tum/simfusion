# SimFusion
Replication package of the paper "SimFusion: Multi-Fidelity Simulation-based Testing of Automated Driving Systems"

SimFusion is validation framework for Automated Driving System which combines low-fidelity simulation with high-fidelity simulation to efficiently evaluate systems.
